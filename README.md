# PromptToCode
将Prompt文件转换成代码并自动执行(一分钟拿AI写出来的毕设)
# 使用方法
安装Python环境，安装openai库<br/>
```
pip install openai
```
用法:<br/>
python main.py <Prompt File> <API_BASE> <API_KEY> <Model><br/>
例子:<br/>
python main.py helloworld.prompt https://api.moonshot.cn/v1 sk-xxx moonshot-v1-8k<br/>
